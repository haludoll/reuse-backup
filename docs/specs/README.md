# ReuseBackup 外部仕様書

## 概要

ReuseBackupは、古いiPhoneを写真・動画のローカルバックアップサーバーとして再利用する革新的なiOSアプリケーションエコシステムです。このドキュメントでは、ユーザーから見える外部仕様について説明します。

- **[クライアントアプリ仕様](./client/README.md)** - ReuseBackupClient（新しいiPhone用）

### 共通仕様

## 対応バージョン

- **サーバーアプリ**: iOS 15.0+
- **クライアントアプリ**: iOS 17.0+  
- **API バージョン**: v1.0.0
